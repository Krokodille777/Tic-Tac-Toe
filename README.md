# Tic-Tac-Toe


A simple Tic-Tac-Toe game built with HTML, CSS, and JavaScript. Play against a basic AI opponent.

## Features
- Interactive Tic-Tac-Toe board using HTML5 Canvas.
- Simple AI opponent that makes random moves.
- Responsive design for various screen sizes.
- Start and close game functionality.

## How to Play
1. Open `index.html` in your web browser.
2. Or just click the link in the GitHub Pages section below.
3. Click the "Start Game" button to begin playing.
4. Click on the canvas to place your "X". The AI will automatically place an "O".
5. The game will announce the winner or if it's a draw.
6. Your sides will be automatically swapped for the next games.
7. Click the "Close Game" button to exit the game board.
